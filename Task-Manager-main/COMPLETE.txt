================================================================================
                        QUICK COMMANDS
================================================================================

Backend:
   cd backend
   npm install
   npm run dev              (start dev server)
   npm run build            (build for production)
   npm start                (run production)

Frontend:
   cd frontend
   npm install
   npm run dev              (start dev server)
   npm run build            (build for production)
   npm start                (run production)

Database:
   npm run prisma:generate  (generate client)
   npm run prisma:migrate   (run migrations)
   npx prisma studio       (view database)
